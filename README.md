# Dellulu Finance Dashboard

Real-time e-commerce analytics dashboard powered by Supabase.

## Live Dashboard

[View Dashboard](https://admindellulu.github.io/dellulu-dashboard/)

## Features

- 7 analytics tabs: Overview, Orders, Finance, Ads, Shipping, Settlements, Data Quality
- Real-time Supabase connection
- Event-driven data sync (Shopify + Razorpay webhooks)
- Daily/Weekly/Monthly metric toggles per chart
- Mobile responsive

## Tech Stack

- Single-file HTML/CSS/JS
- Chart.js for visualizations
- Supabase REST API for data
- GitHub Pages for hosting
